# Nordic Operations Hjemmeside v2.0

Produktionsklar frontend-arkitektur optimeret til udrulning på **GitHub Pages**.

## Funktioner
- **Målgruppe & B2B Fokus**: Fjernet generiske salgstal til fordel for reelle compliance-fordele (NIS2, GDPR, CRA).
- **Modulær opbygning**: Klar til fremtidige udvidelser (undersider til produkter, ressourcecenter og kundelogin).
- **Progressive Web App (PWA)**: Fuldt integreret service worker og manifest-konfiguration.
- **Europæisk Design-niveau**: Stripe/Linear inspireret minimalistisk mørkt tema med limegrønne kontraster.
